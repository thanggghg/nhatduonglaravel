# MVP Checklist

## 1. Chức năng bắt buộc cho bản đầu tiên

```text
[ ] Trang chủ
[ ] Banner trang chủ
[ ] Danh sách tuyến xe nổi bật
[ ] Trang danh sách tuyến xe
[ ] Trang chi tiết tuyến xe
[ ] Trang lịch trình tham khảo
[ ] Trang danh sách tin tức
[ ] Trang chi tiết tin tức
[ ] Trang giới thiệu
[ ] Trang hướng dẫn đặt vé
[ ] Trang liên hệ
[ ] Nút đặt vé redirect sang tool bên thứ ba
[ ] Admin quản lý banner
[ ] Admin quản lý bài viết
[ ] Admin quản lý danh mục bài viết
[ ] Admin quản lý tuyến xe
[ ] Admin quản lý lịch trình
[ ] Admin quản lý trang tĩnh
[ ] Admin quản lý FAQ
[ ] Admin quản lý cấu hình website
```

## 2. Chức năng nên có nhưng có thể làm sau

```text
[ ] Tracking lượt click đặt vé
[ ] Dashboard thống kê click
[ ] Tối ưu SEO nâng cao
[ ] Sitemap XML tự động
[ ] Google Analytics
[ ] Facebook Pixel
[ ] Live chat
[ ] Đa ngôn ngữ
[ ] Popup khuyến mãi
[ ] Đăng ký nhận tin
```

## 3. Chức năng không làm trong MVP

```text
[ ] Chọn ghế
[ ] Giữ ghế
[ ] Thanh toán online trực tiếp trong Laravel
[ ] Tạo mã vé
[ ] Quản lý booking realtime
[ ] Hoàn vé
[ ] Hủy vé
[ ] Quản lý tài xế
[ ] Quản lý xe chi tiết
```

## 4. Tiêu chí nghiệm thu MVP

```text
[ ] Khách có thể xem thông tin tuyến xe.
[ ] Khách có thể xem lịch trình tham khảo.
[ ] Khách có thể đọc tin tức.
[ ] Khách có thể bấm đặt vé và được chuyển sang tool bên thứ ba.
[ ] Admin có thể quản lý toàn bộ nội dung chính.
[ ] Website hiển thị tốt trên mobile.
[ ] Website có SEO title và description cơ bản.
[ ] Form liên hệ hoạt động.
```

## 5. Ưu tiên triển khai

Thứ tự ưu tiên:

```text
1. Admin CMS
2. Trang chủ
3. Tuyến xe
4. Tin tức
5. Trang tĩnh
6. Tích hợp đặt vé bên thứ ba
7. SEO
8. Tracking
```
