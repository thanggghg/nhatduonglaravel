# Task Management - Nhat Duong Laravel Project

Quản lý các task triển khai dự án Website bán vé xe theo kế hoạch.

## ⚠️ DESIGN SYSTEM REQUIREMENT

**CRITICAL:** Tất cả thiết kế phải tuân theo [DESIGN.md](../DESIGN.md)

### Design Guidelines
- **Colors:** Brand Green (#0b7f42), Brand Gold (#fbb116)
- **Typography:** Inter font family
- **Spacing:** Base unit 8px, Section gap 64px, Element gap 24px
- **Border Radius:** 8px (buttons/inputs), 16px (cards), 999px (badges)
- **Shadows:** Green-tinted rgba(11, 127, 66, 0.10)
- **Components:** Follow specifications in DESIGN.md

### Before Starting Any UI Task:
1. Read DESIGN.md thoroughly
2. Understand color usage: Green for primary actions, Gold for secondary/highlights
3. Use exact hex codes, spacing values, and typography scale
4. Reference component specifications for buttons, cards, badges, inputs
5. Check Do's and Don'ts section

## Trạng thái task

- `[ ]` - Chưa bắt đầu
- `[~]` - Đang làm
- `[x]` - Hoàn thành
- `[-]` - Tạm hoãn

## Quy trình làm việc

1. Đọc file task tương ứng với giai đoạn hiện tại
2. Cập nhật trạng thái task khi bắt đầu làm
3. Đánh dấu hoàn thành khi xong
4. Commit và push lên GitHub
