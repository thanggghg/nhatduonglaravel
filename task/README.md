# Task Management - Nhat Duong Laravel Project

Quản lý các task triển khai dự án Website bán vé xe theo kế hoạch.

## Cấu trúc thư mục task

```text
task/
├── README.md
├── phase-1-project-initialization.md
├── phase-2-admin-cms.md
├── phase-3-frontend-ui.md
├── phase-4-booking-integration.md
├── phase-5-seo-tracking.md
├── phase-6-testing-deployment.md
└── mvp-tasks.md
```

## Trạng thái task

- `[ ]` - Chưa bắt đầu
- `[~]` - Đang làm
- `[x]` - Hoàn thành
- `[-]` - Tạm hoãn

## Quy trình làm việc

1. Đọc file task tương ứng với giai đoạn hiện tại
2. Cập nhật trạng thái task khi bắt đầu làm
3. Đánh dấu hoàn thành khi xong
4. Commit và push lên GitHub
